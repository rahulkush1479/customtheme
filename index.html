Get started

1. WordPress Theme Essential Files

1. style.css (Theme Info & Styles)

style.css -> This file contains the theme information and styles.


/*
Theme Name: My Custom Theme  
Theme URI: https://example.com  
Author: Your Name  
Author URI: https://yourwebsite.com

Version: 1.0  
*/

2. index.php (Main Template File)

index.php: The main template file. For now, you can add

 <!DOCTYPE html>  
 <html <?php language_attributes(); ?>>  
 <head>  
     <meta charset=<?php bloginfo('charset'); ?>>  
     <title><?php bloginfo('name'); ?></title>  
     <link rel="stylesheet" href=<?php bloginfo('stylesheet_url'); ?>>  
     <?php wp_head(); ?>  
 </head>  
 <body <?php body_class(); ?>>  
     <h1><?php bloginfo('name'); ?></h1>  
     <p><?php bloginfo('description'); ?></p>  
     <?php wp_footer(); ?>  
 </body>  
 </html>

3.Theme Activation

  • Log in to your WordPress admin dashboard.
  • Navigate to Appearance > Themes.
  • You should see your theme listed. Activate it.

4.Add Template Files

Depending on your site's structure, you’ll need various template files to control different pages:

  • header.php: Contains the code for the site’s header.
  • footer.php: Contains the code for the site’s footer.
  • sidebar.php: If your theme has a sidebar. - Opotionnsl
  • page.php: Template for standard pages.
  • single.php: Template for single posts.

functions.php: A file where you’ll register theme features like menus, widgets, and more. php 


function my_custom_theme_scripts() {  
wp_enqueue_style('style', get_stylesheet_uri());  
}  
add_action('wp_enqueue_scripts', 'my_custom_theme_scripts'); 

(for setup ----)

function my_custom_theme_setup() {  
register_nav_menus(array(  
'primary' => __('Primary Menu', 'my-custom-theme'),  
));  
}  
add_action('after_setup_theme', 'my_custom_theme_setup');

6. Create a Custom Homepage

To create a custom homepage, you can create a front-page.php template.

7. Dynamic Content with WordPress Loop

Use WordPress’s loop to display posts or content dynamically:


<?php if ( have_posts() ) : ?>
<ul>
<?php while ( have_posts() ) : the_post(); ?>
<li>
<h2><a href="<?php the_permalink(); ?>"> <?php the_title();?>
</a> </h2>
<p><?php the_excerpt(); ?></p>
</li>
<?php endwhile; ?>
</ul>
<?php else : ?>
<p>No posts found.</p>
<?php endif; ?>

8. Add Custom CSS and JavaScript

    You can add custom CSS and JavaScript by enqueueing them in functions.php: 



function theme_custom_scripts() {  
wp_enqueue_style('main-css', get_template_directory_uri() . 
'/assets/css/main.css');  
wp_enqueue_script('custom-js', get_template_directory_uri() .
'/assets/js/custom.js', array('jquery'), null, true);  
}  
add_action('wp_enqueue_scripts', 'theme_custom_scripts');

// Register Custom Post Type
function create_books_post_type() {
 register_post_type('book', array(
     'labels' => array(
         'name' => __('Books'),
         'singular_name' => __('Book')
     ),
     'public' => true,
     'has_archive' => true,
     'supports' => array('title', 'editor', 'thumbnail'),
     'menu_icon' => 'dashicons-book', 
     // Book icon in the admin menu
 ));
 add_action('init', 'create_books_post_type');
 function create_book_genre_taxonomy() {
     register_taxonomy('book category', 'book', array(
         'labels' => array(
             'name' => __('Book Categories'),
             'singular_name' => __('Book Category')
         ),
         'public' => true,
         'hierarchical' => true, // Set to true for 
         category-like behavior (parent-child)
     ));
 }
 add_action('init', 'create_book_genre_taxonomy');
